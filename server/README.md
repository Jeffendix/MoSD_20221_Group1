# Travelo Server




## Installation

Install mongodb compass 
```console
https://www.mongodb.com/try/download/shell
```


Install library:

```console
$ npm install
```

Init database:

```console
$ npm run seed
```

## Start Server


```console
$ npm run dev
```
